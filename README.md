# Photogrammetric Computer Vision

This repository contains my exercises and Python implementations for the course **Photogrammetric Computer Vision**.

## Contents
- **Exercise 01**: Homogeneous coordinates and basic projective geometry  
- **Exercise 02**: Projective transformation (homography), rectification, panorama stitching  
- **Exercise 03**: Camera calibration using DLT (projection matrix)  
- **Exercise 04**: Relative orientation of an image pair (epipolar geometry, fundamental matrix)  
- **Exercise 05**: Structure-from-Motion (SfM) pipeline

## Structure
Each exercise is stored in its own folder:
- `ex01_homogeneous_coordinates/`
- `ex02_projective_transformation_homography/`
- `ex03_camera_calibration_dlt/`
- `ex04_relative_orientation_fundamental_matrix/`
- `ex05_structure_from_motion/`

## Tools
- Python
- NumPy
- OpenCV
- Matplotlib

## Notes
This repository is for learning purposes and documents the implementation and results of the course exercises.
